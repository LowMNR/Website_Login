CPU

La CPU (Central Processing Unit) è il nucleo di tutto il sistema di elaborazione. È in grado di leggere i dati e le istruzioni dalla memoria di lavoro per poi eseguirle, una alla volta, con una velocità molto alta. Il funzionamento della CPU può essere riassunto in tre fasi:
Fetch in cui la CPU estrae le istruzioni dalla memoria;
Decode in cui la CPU decodifica ed interpreta le istruzioni. Queste istruzioni sono di solito scritte con linguaggi di programmazione che utilizzano parole tipiche del linguaggio umano (di solito in inglese), mentre la CPU esegue istruzioni scritte in linguaggio macchina, ovvero in binario. Interpretare vuol dire tradurre le istruzioni da linguaggio di programmazione a linguaggio macchina;
Execute in cui la CPU esegue le istruzioni.
La CPU, inoltre, gestisce il trasferimento dei dati tra i vari componenti attraverso i bus di sistema.
